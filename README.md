# FPC Butterflies

A collaborative school project between Columbus School for Girls and Franklin Park Conservatory.

## About This Project

This project explores the fascinating world of butterflies through hands-on learning and research. Students from Columbus School for Girls are working with Franklin Park Conservatory to study butterfly biology, habitats, and conservation.

## Project Goals

- Learn about butterfly life cycles and ecosystems
- Understand the importance of pollinators in our environment
- Document and track butterfly species
- Engage in conservation efforts

## Getting Started

This repository will contain project documentation, research findings, and any code or data related to the butterfly study project.
