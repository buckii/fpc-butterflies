# Blooms & Butterflies — Franklin Park Conservatory

An interactive classroom extension activity about butterflies, hosted as a single-page web app.

## Project Structure

```
index.html   — All slide content and HTML markup
styles.css   — All CSS styles (extracted from inline <style>)
app.js       — All JavaScript logic (extracted from inline <script>)
images/      — Photos and illustrations used throughout the slides
```

## Features

- **Main Menu** — Full-bleed meadow background with semi-opaque content panel
- **Tutorial** — How-to-play instructions
- **Teacher's Guide** — Resources, standards alignment, and classroom project ideas
- **Host Plants Matching** — Interactive plant-to-butterfly matching activity (8 pairs)
- **Camouflage Scenes** — Find the hidden butterfly in 4 different scenes, with hint after 3 wrong clicks
- **Butterfly Gallery** — Learn about 6 species (Monarch, Blue Morpho, Painted Lady, Swallowtail, Red Admiral, Viceroy)

## Running Locally

Open `index.html` directly in a browser — no build step or server required.
