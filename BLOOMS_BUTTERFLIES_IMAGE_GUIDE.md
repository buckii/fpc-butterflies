## Implementation Notes

### HTML Structure
The existing `index.html` likely has a basic structure. Images should be inserted as:

```html
<!-- Background images -->
<div class="slide" style="background-image: url('Blooms and Butterflies images/slide01_07-11_wildflower_meadow_background.jpg')">
  <!-- Slide content -->
</div>

<!-- Content images -->
<img src="Blooms and Butterflies images/slide13_monarch_and_caterpillar_on_milkweed.png" alt="Monarch caterpillar on milkweed">

<!-- Icons/Interactive elements -->
<button class="answer-btn">
  <img src="Blooms and Butterflies images/slide47_48_51_54_57_icon_green_check.png" alt="Correct">
  YES
</button>
```

### Image Path
All images should reference: `Blooms and Butterflies images/[filename]`

### Responsive Sizing
Since all images are max 2000x2000px, they should scale well. Use CSS:
```css
img {
  max-width: 100%;
  height: auto;
}
```
